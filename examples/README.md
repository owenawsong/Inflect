# Examples

This directory is reserved for lightweight public examples.

Planned examples:

- `quickstart_local.ps1`
- `voice_clone_minimal.py`
- `render_sample_gallery.py`
- `enhance_audio.py`

Large audio files, private reference voices, generated datasets, and checkpoints should not live here. Public samples should be linked from a release page or Hugging Face dataset/model card instead.
